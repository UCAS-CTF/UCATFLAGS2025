INTRO Program Coder

PPC (Professionally Program Coder) 考察快速编程解决问题的能力。你需要结合算法功底与编程技巧，自动化地处理数据或与服务器交互。

FOUNDATION

编程基础

  • >> Input/Output: 快速IO (scanf/printf), 格式化
  • >> Types: 高精度运算 (BigInt), 溢出处理
  • >> Interaction: Socket 编程 (pwntools/nc)
  • >> Scripting: 正则表达式, 字符串/字节流处理

ALGORITHMS

数据结构

  • >> Linear: 栈, 队列, 链表
  • >> Tree: 二叉搜索树, 堆 (Heap), 线段树
  • >> Graph: DFS/BFS, 最短路 (Dijkstra), 最小生成树
  • >> Method: 动态规划 (DP), 二分查找, 贪心
