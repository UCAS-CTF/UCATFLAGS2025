INTRO Program Coder

PPC (Professionally Program Coder) 考察快速编程解决问题的能力。不仅需要扎实的算法功底，还要有快速编写脚本处理数据的技巧。

FOUNDATION

语言基础

  • Input/Output: 快速IO (scanf/printf)
  • Types: 熟悉各种数据类型溢出
  • Logic: 递归, 循环控制
  • Scripting: Python requests/socket

ALGORITHMS

数据结构

  • Linear: 栈, 队列, 链表
  • Tree: 二叉树, 堆 (Heap)
  • Graph: DFS, BFS, 最短路
  • Other: 排序, 二分查找, 哈希
