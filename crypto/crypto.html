INTRO Mathematics & Logic

Crypto 侧重于古典密码与现代密码学的算法原理。你需要深厚的数学基础，去破解那些看似杂乱无章的密文。

ALGORITHMS

常见算法

  • >> 对称密码: AES, DES, RC4
  • >> 非对称密码: RSA, ECC, ElGamal
  • >> 哈希函数: MD5, SHA-1, SHA-256
  • >> 编码方式: Base64, Hex, ASCII

TARGETS

攻击对象

  • >> 分组模式: ECB/CBC 翻转攻击
  • >> RSA攻击: 共模, 小指数, 维纳攻击
  • >> 离散对数: DLP 问题求解
  • >> 随机数: 伪随机数预测 (PRNG)
