INTRO Web Exploitation

CTF Web 题目主要涉及网络安全和 Web 应用的漏洞分析与利用。你需要像黑客一样思考，寻找 HTTP 协议、服务器配置或代码逻辑中的破绽。

CORE CONCEPTS

基础架构

  • >> HTTP协议: 请求/响应, 状态码
  • >> Web服务器: Apache, Nginx
  • >> 脚本语言: PHP, Python, JS
  • >> 数据库: SQL 语法, 权限

VULNERABILITIES

常见漏洞

  • >> 注入攻击: SQLi, 命令注入
  • >> 客户端: XSS, CSRF, CORS
  • >> 服务端: SSRF, 文件上传
  • >> 认证授权: JWT, OAuth
